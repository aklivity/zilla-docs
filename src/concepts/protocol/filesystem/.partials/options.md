### options

| Property | Type                          | Description                                            | Required | Default Value |
| -------- | ----------------------------- | ------------------------------------------------------ | -------- | ------------- |
| location | `string`                      | File system URI or directory name with trailing slash. | No       |               |
| symlinks | `enum` [ `follow`, `ignore` ] | How to treat symbolic links.                           | No       | `ignore`      |
